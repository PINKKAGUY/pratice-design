# Design Showcase

A landing page that demonstrates front-end design skills through its own execution. No frameworks, no libraries — just vanilla HTML, CSS, and JavaScript.

**[View Live Site](https://pinkkaguy.github.io/pratice-design/)**

## What It Showcases

- **Splash Screen** — Typewriter animation with realistic typo/correction effects
- **Typography** — Type hierarchy, font scale, and spacing system built on Inter
- **Layout & Grid** — CSS Grid compositions with asymmetric card layouts
- **Complex Components** — Interactive calendar, kanban board, dashboard with animated charts, and chat interface
- **Micro-interactions** — Buttons, hover states, floating label inputs, toggles, and checkboxes
- **CSS Animations** — Morphing shapes, SVG path drawing, orbit systems, staggered reveals

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, Grid, Flexbox, keyframe animations)
- Vanilla JavaScript (IntersectionObserver, drag & drop API)
- Zero dependencies

## Features

- Animated splash screen with typewriter effect and simulated typos
- Scroll-triggered reveal animations
- Tab-based component switcher with fade transitions
- Drag & drop kanban board
- Interactive calendar with time slot selection
- Live chat demo with auto-replies
- Animated dashboard bar charts
- Fully responsive (mobile-first)
- `prefers-reduced-motion` support
- Keyboard accessible

## Run Locally

Just open `index.html` in a browser. No build step required.
